Getting Started
===============

Integrate Gryffin into your project quickly!


Installation
------------

Install Gryffin via pip with the command:

.. code-block:: console

    $ pip install gryffin

If you are installing from source, simply run pip install from the root directory:

.. code-block::

    /path/to/gryffin/$  pip install .
